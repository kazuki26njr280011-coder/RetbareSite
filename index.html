RetBareServerは、Minecraft Java Edition 1.21.1で動作するマルチプレイサーバーです。BedWars・SkyWars・Duel・サバイバル・OneBlockなど多彩なコンテンツに加え、独自経済、5種のガチャ、Ultra Cosmetics、プレイヤートレード、80種以上のカスタムエンチャント、SmartSpawnerによる自動ファームを搭載。N高グループ生徒コミュニティを中心に運営されています。

01

PVPコンテンツ

BedWars・SkyWars・Duel(8種目)。サーバー独自のアイテム・KITが多数。勝てば即報酬。

02

サバイバル経済

素材売却・オークション・PVP報酬・ログインボーナス・トレード。Vault基盤で安定運用。

03

独自システム

SmartSpawner、MobCoin、Ultra Cosmetics、リカバリートーテム、3×3ツルハシ。

RetBareServerのPVPは、サーバー独自のアイテムとKITによって他では味わえない戦闘体験を提供します。勝てば即座にサバイバルで使える円が手に入るため、PVPをやりこむほど経済的にも有利になります。

CHAMP MODE

BedWars

チームのベッドを守りながら相手のベッドを破壊するチーム対抗戦。TNT羊などサーバー独自のアイテムが多数登場。勝利で1,028円、ベッド破壊で1,000円の報酬を獲得。新マップ「BIG」は最大64人対応。

BATTLE ROYALE

SkyWars

空島から始まるバトルロイヤル形式のPVP。最後の1人が勝者。サーバー独自のKIT制度が豊富で、/ss から1個5,000円で新キットを購入できます。日本語化済み。

SOLO MODE

OneBlock

足元のたった1ブロックから始まる世界。ブロックを壊すたびに新しいブロックが出現し、集めた素材で拠点を拡大していきます。/rmenu から参加可能。

MAIN MODE

サバイバル

通常のMinecraftサバイバルに、独自経済・ガチャ・オークション・MobCoin・SmartSpawnerを加えたメインモード。ネザー・エンド開放済み。

RetBareServer専用のDuelシステム。8種目のPVPで1vs1の真剣勝負ができます。マッチング形式で相手を見つけ、勝敗を決定。UHC FFA以外はすべて1vs1、UHC FFAのみ乱闘形式です。

参加方法

/warp duel でDuelロビーへ移動 → /duel でGUIを開く → 種目を選んでマッチング開始。すべての操作はGUIで完結します。アリーナからの退出もGUIから可能。Duel日本語化済み。

Ultra Hardcore。自然回復なしの真剣勝負。相手を倒したら勝ち。

FFA

乱闘形式のUHC。複数プレイヤーが同時に戦い、最後の1人を目指す。

1V1

1V1

デバフなしのポット戦。純粋なPotスキルを競う種目。

通貨単位は「円」。Vaultベースの経済基盤で運用されており、所持金は /money で確認できます。お金の稼ぎ方は4つ、使い道はショップ・5種のガチャ・オークション・プレイヤートレードと豊富です。

METHOD 01

Shopで素材を売る

Shop内の蜂のNPCに右クリックすると交易画面が開き、アイテムを売却できます。例: オークの原木で12円。「YOUPRICE: [金額]」が受取額です。

METHOD 02

オークションで売る

/ah で出品。他プレイヤーに購入されたら売値が手に入ります。一定時間で期限切れになりますが、回収可能です。

METHOD 03

BedWarsで稼ぐ

勝利で1,028円、ベッド破壊で1,000円のボーナス。PvPで遊びながら効率よく稼げます。月間ランキング上位には豪華報酬も。

METHOD 04

ログインボーナス

ログインするだけで一定額がもらえる仕組み。毎日少しずつ稼ぎたい人向け。

RetBareServerには5種類のガチャが存在し、それぞれ異なる報酬が排出されます。マジック・エンチャントガチャは直接開封、ダイヤ・ネザライト・スポナーガチャはキーで開封する形式です。

UNIQUE FEATURE 01

SmartSpawner
放置で素材が貯まる最強自動ファーム

通常のスポナーの超強化版。実際にモブをスポーンさせず、ドロップアイテムとXPを仮想的に自動生成します。サーバーにAI処理の負荷がかからず、ラグが発生しません。右クリックでGUIを開き、溜まったアイテムを一括回収。ホッパーで自動搬出も可能。スポナー同士のスタックで効率アップ。

UNIQUE FEATURE 02

MobCoin
モブを倒して貯める特殊通貨

通常の円とは別系統の通貨。モブを倒すと獲得でき、専用ショップでSmartSpawnerなどの特殊アイテムと交換できます。/mobcoins で残高確認、/mobcoins shop でショップを開きます。プレイヤー間の送金、TOP10ランキング表示にも対応。

UNIQUE FEATURE 03

カスタムエンチャント
80種以上のバニラライク拡張

矢がエンダーパールに変わる弓用エンチャント、雷を落とすエンチャント、ドラゴンの炎、TNT射出、ホバー効果、一括採掘など80種以上。すべて日本語説明付きで、初見でも内容がわかります。エンチャントガチャから入手。

サーバーロビーで使えるコスメティック機能。Ultra Cosmeticsプラグインを採用しており、ガジェット・ペット・マウント・モーフ・ハット・エモートなど膨大なカテゴリのコスメが用意されています。Duelアリーナ内では使用できません。

UNIQUE FEATURE 04

Ultra Cosmetics
ロビーが自分だけの遊び場に

エンダードラゴンに乗って空を駆けたり、TNTを投げまくったり、ペットを連れ歩いたり、別のモブに変身したり。コスメ専用のガチャがあり、Retbare通貨2,000円で1回引けます。コスメの装備・解除はすべて /uc menu から行えます。

装備中のコスメをすべて解除するには /uc clear、モーフ変身中の自分の姿を確認するには /uc selfview を使用してください。

NEW FEATURE

プレイヤートレード
直接交換で大富豪を狙え

よくある質問をまとめました。ここに載っていない質問はDiscordでお気軽にどうぞ。

サーバーへの参加方法は？
Minecraft Java版1.21.1を起動し、マルチプレイからサーバーを追加。アドレス欄に 110.67.56.168:25565 を入力して接続してください。
統合版でも遊べますか？
統合版（アドレス: 110.67.56.168:19132）からも接続可能ですが、現在多数の不具合があるため、Java版での接続を推奨しています。統合版プレイヤーは名前の先頭にドット（.）がつきます。
Duelはどうやって遊ぶ？
/warp duel でDuelロビーへ移動し、/duel でGUIを開きます。UHC・UHC FFA・Sword・Axe・Pot・Sumo・Mace・NoDebuffの8種目から選んでマッチング。UHC FFA以外は1vs1形式です。
コスメティックの使い方は？
サーバーロビーで /uc menu を実行するとコスメメニューが開きます。ガジェット・ペット・マウント・モーフなどを選んで装備可能。Duelアリーナ内では使用できません。コスメガチャは2,000円で1回引けます。
トレードのやり方は？
/trade <相手の名前> でリクエストを送信。相手が承認するとトレード画面が開き、お互いにアイテムをセットして取引できます。/trade aspect <名前> で拡張トレードも可能。
お金（円）の確認方法は？
/money コマンドで現在の所持金を確認できます。
ガチャはどこで引けますか？
マジック・エンチャントガチャはShopスポーン左のシュルカーボックス5個の場所で直接開封。ダイヤガチャはウーパールーパーのショップ（ネザライトブロック周辺）、スポナーガチャはキーを購入して開封します。
MobCoinって何ですか？
モブを倒すと得られる特殊通貨です。/mobcoins で残高確認、/mobcoins shop でSmartSpawnerなどを購入できます。
SmartSpawnerとは？
実際にモブをスポーンさせず、ドロップアイテムとXPを仮想生成するスポナー。サーバー負荷ゼロでアイテムが貯まる最強の自動ファームを作れます。
リカバリートーテムって？
インベントリに入れた状態で死亡すると、トーテム自体は消費されますが、他の全アイテムが保持される独自アイテム。Shopで2,000円、マジックガチャでも入手可能です。
困ったときは？
Discordサーバーでチケットを作成し、スタッフに相談してください。
